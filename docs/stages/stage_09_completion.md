## Customer rules

- Physical summaries require **confirmed** (non-revoked) target identity
- Sufficient coverage required; low coverage → `not_evaluable` / partial — **not** inactivity
- No coverage extrapolation to full-match claims
- Penalty-area occupancy ≠ ball touch / possession / event
- Sprint / activity definitions are **project_generated** (Opta-style thresholds only) — **not** official Opta data
- Real football match accuracy is **not** validated

## Explicitly not delivered in Stage 9

- Pass / dribble / duel / turnover events
- Human–ball possession
- Box touch / shot events
- Final unified customer analysis visual
- Reviewed ground-truth accuracy claims
